关于TEDx高新路

作为TEDx 的新生力量，TEDx高新路是在中国的、西北的、唯一的由青年人全程参与组织的TEDx 的最高级别执照活动。

概述

TEDx高新路已于2017 年4 月正式得到官方授权，开始它在西安的各项活动。两年来，TEDx高新路保持初心，传承TED “IdeasWorth Spreading” 精神，通过沙龙、大会等活动，用年轻人的想象力和行动力凝聚古都西安各处的优秀人士，力求给西安大众一次精神的洗礼，传播优秀思想。

组织

TEDx高新路是面对全西安市市民的一次大型公益演讲活动。团队均由不超过20 岁的青年组成，是向西安、向全国展示西安青年形象的一次大型活动。

宗旨

用青年人的视角，去审视西安的发展、未来、科技、教育、传奇；用青年人的想象力、行动力凝聚古都西安各处的优秀人士，让青年人的优秀思想给西安大众一次精神的洗礼。
