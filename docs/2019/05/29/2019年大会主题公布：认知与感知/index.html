2019年大会主题公布：认知与感知

前言 始于2017

TEDx高新路大会的背后，是我们始终不变的理想。那就是将时代的活力与思想融为一体。本届大会将这一概念带上了更高境界，认知与感知。

在这个充满变数的世界，谁不想紧随时代的脚步？关键就在于如何认识这个世界，去感其变化，知其规律。

本届大会将着力于对世界的理解。从为孩子创造更好的环境到保护动物，从了解历史到生活美学，旨在让观众体验从感受到感知的美妙进程。

世界选择你，你选择我

用来判断方向的，是认知

认知是一种理性选择。

每一个人来到这个世界，冥冥之中都是世界的选择。自我们出生，就在不断对它进行着自我认知。我们因脸上的阳光温暖，而认知太阳的热；我们因路边的花红柳绿，而认知春天的美；我们因眼里的獠牙锋利，而认知猛兽的凶。认知，是我们将万事万物传递出的信号进行接收并转换成自我印象的过程。它让我们基于理性，判断是奔向小溪还是大海，是爱上银河还是一颗星体。

用来决定选择的，是感知

感知是一种感性加工。

在这个世界上，因际遇不同，经历不同，我们每个人都是自己的哈姆雷特。橙子是甜是酸，蜥蜴是美是丑，香菜是香是臭，每个人的答案都不尽相同。就像伤心的人不听慢歌，心情好的人觉得雨天都不算沉闷，在理性的认知之上，我们总是会因为自己的感情色彩，而对最初的认知做以感性加工，最终，决定我们真正选择的，都是感知。在看过小溪之后，我选择爱上大海；哪怕已见过银河的壮丽，但我仍选择爱上一颗星的孤独。

认知与感知

这五个字就是本届大会的主题

简单来说，认知的过程是主观客观化的过程，既主观和客观同时存在，它就像我们小时候在公园里玩的跷跷板，只要一边加重，它的整体平衡就会被打破。而我们当今就是缺少了绝大多数的客观，反而使主观反入为主，这使经验主义总是在影响着我们的思维判断。

感知是通过感官接受世界传达的信息，认知是通过思维认识万物的本质内涵。感知是一切生物体都具备的能力，而认知却是人的特质。

感知与认知是相对的趋势。曾几何时，我们竟逐渐偏向极端，坚信理论却失去了对生活的感知，亦或是浮于浅表却难以深刻地认知。我们该如何将二者做好平衡？

本届将着重于帮助大家探索自己，以及激起我们对周围世界的思考。亲密与独立和生活美学这两个主题就是在以特殊的角度出发，并且它并不针对于特殊一群体，而是涉及到我们每个人。

主题之美

在2019年初，我们启用了全新的TEDx图标，TEDx高新路。

中文带来了更小的长宽比，这增强了整体的美观性和辨识度。中文名称采用了由Adobe和Google推出的思源黑体。

我们汲取了Behance上优秀作品的灵感，制定了一套标准色和两套LOGO，方圆各一，中英兼备。

在标准字中，中文字体采用方正俊黑。方正俊黑字形颀长，笔形简练，风格简静优雅。英文字体采用Google推出的Barlow. Barlow是一个略圆的，低对比度的Grotesk型字体家族， 灵感源于加州公共设施的视觉风格。

当然上面提到的只是一小部分，我们将会持续推出更多设计。届时，所有设计将于大会当天和所有观众见面。
