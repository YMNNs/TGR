策展团队春季招募

发表于 2019-03-16

前途似海 期待同行

我们，TEDx高新路

TEDx高新路作为陕西省西安市乃至全国唯一一个由高中生参与活动全程策展、举办的独立TEDx团体，TEDx高新路不仅为优秀思想提供传播平台，更多的是想要帮助广大高中生开阔眼界，增加社会实践。TEDx高新路的团队充满着独属于年轻人的纯粹与活力，2017年我们进入大众视野，2018年我们取得了质的进步。在TEDx的道路上，TEDx高新路坚持一步一个脚印，不忘初心，始终致力于为古城西安的人们奉献一场又一场的思想盛宴。

在这里，相似的同龄人们将有机会交流彼此对万事万物的看法，以求用不同的角度和更多的方式去看世界，我们尊重不同，也希冀能带来变革的差异。年轻而又充满活力的我们，期待与每一个有趣的灵魂相遇。前途似海，期待同行。

加入我们的∞个理由

如果你想要与优秀的人进行深层次的交流，TEDx高新路的所有讲者正在期待与你的思想碰撞出火花；如果你想要提升自己的活动策划能力，我们的公关资源和宣传资源将给你提供平台；如果你想要丰富自己的社会实践，多种形式的策划活动将帮助你得到一份优秀的实践报告；如果你想要扩展自己的人际关系，TGR策展团队正在寻找与我们相似的人，你将认识各大高校和各个领域优秀的学生和人士。

入职要求

  1. 你居住在西安市且在西安市的高中就读

  2. 你热爱TEDx

我们提供以下职位

职位 你需要掌握的技能 符合以下条件之一的优先
公关助理 谈判能力、语言组织与表达能力 自带品牌资源、有相关工作经历
微博运营 幽默、语言组织与表达能力 知名博主、小编（非营销号）
视觉设计 审美、动手能力 字体收藏家、Adobe全家桶爱好者

那么，我们开始！

扫描二维码填写申请

然后呢？

我们会认真审阅你的申请（三个工作日内），如果你符合我们的入职条件，你将会收到我们的邮件。接着我们会为你安排关于TEDxRules和职位技能的笔试，并为通过者安排最终的面试，全程不收取任何费用。

为了顺利通过笔试，您可以先学习TEDx Rules.

https://www.ted.com/participate/organize-a-local-tedx-event/before-you-start/tedx-rules

如果您想稍后填写问卷，请记下问卷链接。

https://wj.qq.com/s2/3323942/f81a/

别再犹豫了，只要你富有自信和热情，你就是我们要找的那个人！
